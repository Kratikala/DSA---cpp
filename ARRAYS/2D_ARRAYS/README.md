# 2D Arrays

This folder contains my practice and revision of 2D array concepts and matrix-based problems in C++.

## Topics Covered

- Matrix Basics
- Input & Output
- Row-wise Traversal
- Column-wise Traversal
- Matrix Multiplication
- Spiral Traversal
- Saddle Point
- Matrix Manipulation

## Skills Practiced

- Nested Loops
- Matrix Traversal
- Row & Column Operations
- Index Manipulation
- Dry Running Algorithms
- Time Complexity Analysis

## Complexity

Most problems in this folder are solved using:

- Time Complexity: O(rows × columns)
- Space Complexity: O(1) *(excluding input storage)*

## Learning Objectives

- Understand how matrices are stored in memory.
- Master row-wise and column-wise traversal.
- Develop problem-solving skills using nested loops.
- Build a strong foundation for advanced matrix and graph problems.
