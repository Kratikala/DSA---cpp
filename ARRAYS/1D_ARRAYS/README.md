- Linear Search

- Finding Maximum & Minimum
- Frequency Counting

## Goal

The objective of this folder is to build a strong foundation in array manipulation, problem-solving, and algorithmic thinking before moving on to advanced data structures like Linked Lists, Trees, and Graphs.
